  Exploration, Infrastructure, Prepare for Attack, Attack, 
  Secure Holdings After Attack, Incursion, Prepare for Defense, 
  Defend (Short Term), Defend (Long Term), Not Connected 

Tech Area Name:
  The name of the tech area.
  This project will only be added if:
    This Tech Area is visible.
    The Tech Area Level has not been achieved yet.
    Adding this tech area will not violate the Tech Area Min Percentage
      of any of the existing items.

  Even if an item is marked as 100, it can be with other projects. If there
  are already exisiting projects and they allow another project to be
  added (say 2 projects which each have 33 or 25), then the 100 will be
  added. After that however, no new projects will be added until the 
  100 eventually completes.  

Tech Area Level:
  The level of this tech area that must be obtained before this
  item is complete. To make sure an entire tech area is completed
  to its maximum tech, set this value to 9999.
  
Tech Area Min Percent:
  100 - This needs to be the only research project.
  50  - There can be one other project with this.
  33  - There can be 2 other projects.
  25  - There can be 3 other projects.

  As written above, this is not an absolute rule. It is only enforced by 
  not allowing new projects to be added. The reasoning for this is even if
  you specify that you want a project to be getting 100% of the research
  points, that doesn't mean you want to wait to start it until you complete
  all of the other projects. You want to start it as soon as possible, and
  then when the others complete, it will be the only one present.
